THE FIVE-PHASE GAMEPLAY LOOP
Map → Armada → Approach → War → Resolution — Chapter 2
Loop Overview
Five Phases, One Core Loop
1. MAP PHASE
Macro strategy. Build, expand, manage resources, move fleets, assign Generals. Paradox-style pause. 4 speed settings.
2. ARMADA PHASE
Fleet logistics sub-mode. Assemble invasion force, load troops, customize equipment. Overlays on Map Phase.
3. APPROACH PHASE
The decision point. Fleet arrives at target. Choose: Espionage (intel), Diplomacy (negotiate), or War (attack).
4. WAR PHASE
Auto-battle. Space combat (formations, CP), then ground combat (deployment, cover, garrison). 5–25 min per battle.
5. RESOLUTION
All simultaneous actions resolve. Territory changes. Casualties tallied. XP awarded. Cycle ends. Return to Map Phase.
Map Phase Details
The Game's Home State
SIMULTANEOUS TURNS
All factions act simultaneously. Conflicts trigger interrupts. Paradox-style pause freezes EVERYTHING — plan in peace, execute in real-time. 4 speed settings: 1×, 2×, 4×, Pause.
OPERATIONS = GENERALS
Operations limited by General count. No General assigned = operation can't proceed past planning. Promote more Generals = run more simultaneous fronts. The core bottleneck.
