Diegetic UI — 5 Faction Skins
The UI IS the Game World — Not a Menu Layer
TERRAN LEAGUE
Clean military holographic. Blue-white. Signal static on battle reports. Boot-up text sequences. Panels flicker with scan lines.
ETERNAL SHARDS
Flowing crystal light. Cyan. UI elements move like liquid. Interface predicts clicks — buttons glow before hover reaches them.
SCRAP-HORDE
Cracked industrial display. Orange-red. Screen shakes during explosions. Random sparks. Untranslated error messages in unknown glyph system.
NECRO-LEGION
Nanobot-assembled display. Green-on-black. UI panels assemble from particles. Text etches itself letter by letter, as if inscribed.
UNITY ACCORD
Geometric precision. Gold-white. Panels snap together like magnetic tiles. Perfectly aligned grid. Enemy faction UI bleeds through when viewing captured technology — intentional, unsettling detail.
