Save System
STORY / STANDARD
Unlimited manual saves at any point. Autosave every 3 cycles (configurable). Save file contains complete game state: galaxy, all factions, all units, Archive, Compendium, alignment scores, active operations.
HARDCORE
Ironman — single autosave file, overwritten every cycle. No manual saves. No Account bonuses. Death is permanent and unrecoverable. Archive SD still awards based on battles won before defeat.
