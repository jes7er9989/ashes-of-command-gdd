§21.3 — Commander Abilities REDESIGNED
Now Faction-Specific — General Rank 3+ Unlock
COMMANDER ABILITY SYSTEM — REDESIGNED
The old trait-based commander abilities (War Cry, Fortify, Rally, etc.) have been replaced by the faction-specific General Rank 3+ unlock system above. Each faction now has a unique commander-tier ability that unlocks when the assigned General reaches Rank 3, replacing the generic trait abilities.
See §21.1c faction tables — ability #5 in each faction is the General Rank 3+ unlock.
BUILDING-LOSS RULE
If a building that grants a CP ability is destroyed, the ability becomes LOCKED — unless another building of the same type exists elsewhere in the same system. This creates meaningful defensive decisions around protecting key infrastructure.
Example: Terran Orbital Defense destroyed → Precision Airstrike locked. But if a second Orbital Defense exists in another territory in the same system, the ability stays available.
FACTION CP MODIFIERS — RETIRED
The old shared faction modifier system (Terran +15% Orbital Strike, Shards +3s Energy Shield, etc.) has been retired. Faction-specific differences are now baked directly into each faction's unique ability set. Each faction has 6 completely distinct abilities with their own costs, effects, and unlock paths.
